# Mybnb

#### [WEB Link](http://mybnb-test.eba-vs2apwam.ap-northeast-2.elasticbeanstalk.com/)

#### Description
Cloning Airbnb with Python, Django, Tailwind and Social Login(Giuhub,Kakao)
##
#### Required
  * Python 3.8
```
pip install -r requirements.txt
```

